# 🛒 End-to-End ML Pipeline for Customer Segmentation

This project applies **RFM (Recency, Frequency, Monetary) Analysis** and **K-Means Clustering** to segment customers based on their purchasing behavior. The solution helps businesses identify valuable customer groups and make data-driven marketing decisions.

## 📌 Project Overview

The project follows a complete Machine Learning workflow, from data preprocessing to model deployment using Streamlit.

## 🚀 Features

- Exploratory Data Analysis (EDA)
- Data Cleaning
- RFM Feature Engineering
- Feature Scaling
- Elbow Method for Optimal Clusters
- Silhouette Score Evaluation
- K-Means Clustering
- Customer Segment Analysis
- Business Recommendations
- Interactive Streamlit Web Application

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Plotly
- Matplotlib
- Streamlit
- Joblib

## 📊 Customer Segments
