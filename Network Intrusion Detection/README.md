\# End-to-End ML Pipeline for Network Intrusion Detection



A machine learning web application that detects \*\*BENIGN\*\* and \*\*DrDoS\_DNS\*\* network traffic using a trained Logistic Regression model.



\## Features



\- Network intrusion detection

\- Interactive Streamlit web application

\- Traffic scenario selection

\- Custom feature input

\- Real-time prediction

\- Feature reference section

\- Clean and responsive UI



\## Tech Stack



\- Python

\- Pandas

\- Scikit-learn

\- Streamlit

\- Joblib



\## Machine Learning Workflow



\- Data Preprocessing

\- Exploratory Data Analysis (EDA)

\- Feature Scaling

\- Logistic Regression

\- Model Evaluation

\- Model Serialization

\- Streamlit Deployment
