# ✈️ Flight Price Prediction

A Machine Learning project that predicts flight ticket prices based on flight details using a Random Forest Regressor. The project includes data preprocessing, model training, evaluation, and an interactive Streamlit web application.

## 🚀 Features
- Exploratory Data Analysis (EDA)
- Data Preprocessing & Feature Engineering
- Multiple Regression Models
- Best Model: Random Forest Regressor
- Interactive Streamlit Web App
- Real-time Flight Price Prediction

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib
- Matplotlib
- Seaborn

## 📊 Models Trained
- Linear Regression
- Polynomial Regression
- Decision Tree Regressor
- **Random Forest Regressor (Best Model)**
